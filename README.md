# QuickPlan – Agenda Web de Eventos del Centro

QuickPlan es una aplicación web en React para consultar y organizar eventos del centro educativo. Permite ver eventos, filtrar por categoría, buscarlos por texto, ver detalles y gestionar favoritos.

## Tecnologías
- React
- Vite
- HTML5 / CSS3 / JavaScript

## Instalación y ejecución

1. Clonar o descargar el proyecto.
2. Abrir terminal en la carpeta del proyecto.
3. Instalar dependencias:
npm install
4. Iniciar servidor de desarrollo:
npm run dev
5. Abrir el navegador en la URL que indique Vite (normalmente http://localhost:5173).
